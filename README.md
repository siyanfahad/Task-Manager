Task Manager Web App

A clean and lightweight Task Manager built with **Flask**, **SQLite**, and **HTML/CSS**.

Tech Stack

- Python (Flask)
- SQLite
- HTML & CSS
- JavaScript (basic fetch API)

Features

- Add and delete tasks
- View all tasks
- Basic JSON API (`/tasks`)
- Clean and responsive UI
